# Professional Job Board

A modern, responsive job board application built with HTML, CSS, and JavaScript.

This project demonstrates the development of an interactive employment platform where users can browse, search, and explore job opportunities. Throughout development, multiple versions were created to improve the interface, add new features, and refine the overall user experience.

## Features

- 💼 Browse job listings
- 🔍 Search for jobs
- 🏢 Company information
- 📍 Job location display
- 🏷️ Job category organization
- 📱 Responsive design
- ⚡ Fast and lightweight interface
- 🎨 Clean and modern user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Getting Started

### Clone the repository

```bash
git clone https://github.com/endritngjelina/Professional-Job-Board.git
```

### Run locally

Open the latest HTML file directly in your browser, or launch a local web server.

Using Python:

```bash
python -m http.server
```

Then visit:

```
http://localhost:8000
```

Alternatively, use the **Live Server** extension in Visual Studio Code.
